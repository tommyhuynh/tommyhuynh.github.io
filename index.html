TLDR
October 2015 • Developer • Google Chrome Extension • Cal Hacks
TLDR is a Chrome Extension that allows users to summarize any online article into more digestable form. Includes features such summary length and articles themes. It currently has over 3000 users.

TL;DR

Previous Next

TLDR can be describe as "SparkNotes but for web articles!" It summarize any article by just (1) highlighting the text and (2) clicking "TLDR". Adjust the "Summary Size" settings to scale the length of the summaries to your liking. Perfect for those long news/social articles that no one has time to read! The current version available on the Chrome store currently has over 3000 users. We are updating the application every so often with the most recent update adding a new feature that displays the themes discussed in the article.

My role in this project consisted of building the actual Google Chrome extension itself. This involved learning JavaScript on the spot and utilizing Content Scripts to interact with the web-page in order to pull the highlighted text into our alogorithm. I also helped develop the "popularity" algorithm that determines which sentences best summarizes the article. A quick description of the algorithm:

  1. Parses the article into words and sentences.
  2. Assigns "popularity" points to each occurance of a word and if that word occurs in the title.
  3. Sentence "popularity" is then formed from the sum of those words' values.
  4. Return a certain percanatage of the most highly ranked sentences in chronological order.
